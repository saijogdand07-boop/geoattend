First-time setup
⚙️ Firebase config needed — do this once to enable real-time sync and cloud storage.
1
Go to console.firebase.google.com → Create a project → name it geoattend
2
In Firebase console → Build → Realtime Database → Create database → choose Start in test mode
3
Go to Project Settings (gear icon) → Your apps → click </> Web → Register app → copy the config values below
4
In Realtime Database → Rules tab → replace rules with: {"rules":{".read":true,".write":true}} → Publish
